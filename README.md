## 🏆 Key Achievements

我致力於解決「跨系統資料斷層」與「人工效率瓶頸」，以下是我的三大代表作：

### 1. AI 名片識別系統
*   **痛點：** 傳統 OCR 對複雜排版辨識率低，人工建檔耗時。
*   **解法：** 開發並串接 **Google Gemini API**，利用 LLM 語意理解能力提取資訊，自動寫入資料庫。
*   **成果：** 
    *   每日自動處理 **500+** 張名片，準確率達 **95%**。
    *   系統穩定高效，**已成功外售給其他企業使用**。

### 2. 極致效率：營運報表自動化
*   **痛點：** 月報表與獎金結算涉及多來源資料（Excel/ERP），人工彙整耗時且易錯。
*   **解法：** 運用 **Google Apps Script** 與 **VBA** 撰寫自動清洗腳本。
*   **成果：** 
    *   月報表製作時間：**4 小時 ➔ 20 分鐘** (效率提升 90%)。
    *   獎金結算流程：**2 個工作天 ➔ 30 分鐘內**。

### 3. 採購自動化閉環系統
*   **痛點：** ERP 庫存與電商平台數據不同步，導致缺貨或超賣。
*   **解法：** 
    *   串接 **億看 (Ecount) ERP API** 實現庫存每日自動更新。
    *   運用 **Python Selenium** 爬取無 API 的外部平台銷貨資料。
    *   建立「採購預測模型」，依歷史數據自動建議下單量。
